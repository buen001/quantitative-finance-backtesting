Este projeto é uma ferramenta robusta de **Finanças Quantitativas** desenvolvida para simular estratégias de investimento em ativos reais. O sistema automatiza a coleta de dados, processa sinais de compra/venda e gera dashboards interativos.

## 🚀 Funcionalidades
- **Conexão Global:** Suporte a ações (B3 e NYSE), ETFs e Criptomoedas via Yahoo Finance.
- **Inteligência de Mercado:** Diferenciação automática de volatilidade para mercados 24/7 (Cripto) e mercados tradicionais.
- **Rigor Estatístico:** Cálculos precisos de Índice Sharpe, Drawdown Máximo e Retorno Anualizado.
- **Dashboard Interativo:** Visualização em 4 painéis utilizando Plotly.
